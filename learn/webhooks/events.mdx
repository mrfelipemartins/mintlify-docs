## `qr`

Event emitted when a new QR Code is received

<Accordion title="Sample Request Body">
  ```json
{
  "event": "qr",
  "channel_id": "47950b36-bb0d-4f5b-9696-c1ae5b3462a8",
  "data": {
    "qr": "qrcode string"
  },
  "timestamp": "1700095512291"
}
  ```
</Accordion>

## `message_ack`

Event emitted when a message acknoledgement status is updated

| Ack | Description |
| --- | --- |
| 0 | Pending |
| 1 | Sent |
| 2 | Received |
| 3 | Read |
| 4 | Played (Only for audios) |


<Accordion title="Sample Request Body">
  ```json
{
    "data": {

}
  ```
</Accordion>
